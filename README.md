# 🎬 Netflix Content Analytics using SQL

## 📖 Overview
This project analyzes Netflix Movies and TV Shows data using SQL to uncover valuable business insights. The analysis focuses on content distribution, trends, ratings, genres, countries, and release patterns using real-world SQL queries.

## 🚀 Features
- Data Exploration and Analysis
- Business-Oriented SQL Queries
- Content Trend Analysis
- Genre & Rating Analysis
- Country-wise Content Distribution
- Movies vs TV Shows Comparison

## 🛠️ Technologies Used
- SQL
- PostgreSQL
- GitHub

## 📊 Key Business Questions Solved
1. Count total Movies and TV Shows.
2. Find the most common rating for content.
3. Identify top content-producing countries.
4. Analyze content added over the years.
5. Find the longest movie.
6. Compare Movies and TV Shows distribution.
7. Discover the most popular genres.
8. Analyze release year trends.
9. Find top directors and actors.
10. Generate business insights using advanced SQL queries.

## 💡 SQL Concepts Applied
- SELECT Statements
- Filtering (WHERE)
- GROUP BY & ORDER BY
- Aggregate Functions
- CASE Statements
- JOINS
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
